Stylus build instructions (MacOS)

CommandLineTools

CommandLineTools must be installed. If these are already installed, the following command should output the path to CLT.

$ xcode-select -p
/Library/Developer/CommandLineTools

In order to install these, run.

xcode-select --install

Go

Go installation instructions: https://go.dev/doc/install

$ go version
go version go1.21.9 darwin/arm64

If using brew, make sure the latest version (go@1.21) is installed and linked.

brew install go@1.21
brew unlink go@X.XX # if any previous version is installed
brew link go@1.21

Rust

Rust installation instructions: https://www.rust-lang.org/tools/install

Update Rust, install nightly, default to stable and add the wasm targets.

rustup update
rustup toolchain update nightly
rustup default stable
rustup target add wasm32-wasi
rustup target add wasm32-unknown-unknown

Docker

In order to run the tests, Docker must be installed and running. Docker installation instructions: https://docs.docker.com/get-docker/

Stylus

git clone https://github.com/OffchainLabs/stylus
cd stylus
git submodule update --init --recursive --checkout
make clean && make wasm-ci-build

Optional? For running tests LLVM

brew install llvm
brew link llvm
