RSA

dec(enc(M))=Mdec(enc(M)) = Mdec(enc(M))=M

A trap-door one-way function allows a message MMM to be encrypted as cyphertext CCC given a public (known) key / encryption method (encencenc). CCC is only able to be deciphered in a reasonable time if the private key (decdecdec) is known.

First, assume that MMM can be represented as an integer ∈[0,n−1]\in [0, n-1]∈[0,n−1] (if not, break it into blocks). Encrypting/Decrypting MMM with the public key (eee, nnn) ∈N+\in \mathbb N^+∈N+ works by:

Encrypting MMM

C=enc(M)=Me(mod  n)C = enc(M) = M ^e \quad (\text {mod} \; n)C=enc(M)=Me(modn)

Decrypting CCC

dec(C)=Cd(mod  n)dec(C) = C ^d \quad (\text {mod} \; n)dec(C)=Cd(modn)

To use RSA, one must find ppp and qqq, two very large, random prime numbers.

Theorem states

The security of the scheme relies on the fact that it is computationally infeasible to derive ppp and qqq from nnn.

Fermat's little theoreom

Fermat's little theoreom states that

ap≡a(mod  p)a^p \equiv a \quad (\text {mod} \; p)ap≡a(modp)

for any integer aaa and prime ppp.

If a≢0 (mod  p)a \not\equiv 0 \ (\text {mod} \; p)a≡0 (modp), then

a(p−1)≡1(mod  p)a^{(p - 1)} \equiv 1 \quad (\text {mod} \; p)a(p−1)≡1(modp)

Using Fermat's little theorem in equation (1)(1)(1), we can conclude that

Med=M(Mk(q−1))(p−1)≡M(mod  p)\begin{align*} M^{ed} &= M(M^{k(q - 1)})^{(p - 1)} \\ &\equiv M \quad (\text{mod} \; p) \end{align*}Med​=M(Mk(q−1))(p−1)≡M(modp)​

Using the same logic, we can show that

Med≡M(mod  q)M^{ed} \equiv M \quad (\text{mod} \; q)Med≡M(modq)

Chinese remainder theoreom

The Chinese remainder theorem (adapted to this case) states that:

If x≡M  (mod  p)x \equiv M \; (\text{mod} \; p)x≡M(modp) and x≡M  (mod  q)x \equiv M \; (\text{mod} \; q)x≡M(modq) for two co-prime integers ppp, qqq then x≡M  (mod  pq)x \equiv M \; (\text{mod} \; pq)x≡M(modpq)

Using these two theorems, we conclude that

Med≡M  (mod  n)M^{ed} \equiv M \; (\text{mod} \; n)Med≡M(modn)
