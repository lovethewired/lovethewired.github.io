What is turing completeness?

System/Program has enough power of expression to be equivalent to a turing machine.

Turing machine: Mathematical model describing an abstract machine that can change symbols on a strip of tape according to a table of rules. Capable to implement every computer program.

Some requirements:

  • supports input/output
  • conditional branching
  • infinite loops
  • can use infinite memory (memory limitations often ignored for computer languages)
  • halting problem is unavoidable

What is the halting problem?

Turing used the turing machine to answer the question of whether an algorithm could exist that is able to determine whether a computer program stops at some point or continues on for ever.

Example: Checker app checks whether program M halts

  • approve if M halts
  • reject if M loops

Construct Paradox app: Run Checker app on Paradox app and loop if Check(M) = good. Halt if Check(M) = bad.

  • if Paradox halts => Check(P) = bad, but that means that P should have looped
  • if Paradox loops => Check(P) = good, but that means that P should have halted
