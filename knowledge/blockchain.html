What is a blockchain?

  • Decentralized, public database managed through consensus (mining PoW, PoS)
  • Blockchain state is changed through addition of valid blocks through state transition function st=T(st−1,tx)s_t = T(s_{t-1}, tx)st​=T(st−1​,tx)
  • Block keeps track of hash of parent block, contain list of transactions
  • If data in block is changed, then all following blocks get changed (requires consensus)
  • Transactions must be cryptographically signed by key owners

What is Ethereum?

Ethereum is a decentralized, open, turing complete blockchain that supports smart contracts and recently moved from PoW to PoS. Ethereum is the second biggest blockchain by market cap after Bitcoin. It was conceived by Vitalik in 2013.

How does Ethereum differ from Bitcoin?

What is a UTXO/Account-based chain?

UTXO (Unspent Transaction Output) and Account-based models are different accounting methods to determine coin ownerships.

UTXO's represent unspent transaction outputs similar to IOU notes or cash that can only be spent once. When spending 50witha100 with a 100witha100 bill you give 100whichgetsconvertedtotwo50 which gets converted to two 50whichgetsconvertedtotwo50 bills and the spender and receiver each get one. The blockchain does not store absolute balance of sum of coins. Wallets need to keep track of UTXO notes. These can be combined and spent.

Account based models act similarly to bank accounts.

What is a double spend? How is it prevented?

In ethereum accounts keep track of a nonce (number once) that is increased after every transaction. If a transaction has been previously recorded with the same nonce, the new one will be invalid.
